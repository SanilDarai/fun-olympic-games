== Description ==


DW-CosmosV2 is a fully responsive design. It is a beautiful theme based of space and cosmic inspiration.

DW-CosmosV2 features optional sidebars, center block, and a javascript clock. The PRO version has many more features. Visit our website, http://www.designwicked.com, for more details!
